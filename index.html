My Projects

Audiobook Buddy

Audiobook Buddy is an officially released Android audiobook player now available on the Play Store. I designed, developed, and launched the app entirely on my own — from concept through production — implementing a scalable architecture and advanced metadata system to deliver a polished experience for audiobook enthusiasts.

Technical Overview:

  • My signature metadata enhancement algorithm queries multiple APIs to automatically populate missing cover art and book descriptions. Firebase Cloud Functions handle metadata caching, allowing the app to scale efficiently while avoiding API rate limits.
  • Integrates Google Drive, Dropbox, and Microsoft OneDrive APIs for secure cloud imports that comply with Play Store data safety and OAuth guidelines.
  • Connects to the Internet Archive API to provide access to public domain audiobooks, enabling users to browse by subject and download directly within the app.
  • Supports optional cloud sync and backup via Firebase Authentication and Firestore, storing listening progress, library structure, and bookmarks securely.
  • Built entirely with Jetpack Compose and ExoPlayer for a modern, responsive UI and a full-featured playback experience with background playback support, bookmarking, and 12 handcrafted themes.

Architecture & Tech Stack:

Structured around a clean MVVM architecture with reactive data flow, dependency injection using Hilt, asynchronous operations via Coroutines and StateFlow, and persistence through Room. The design prioritizes modularity, testability, and maintainability for long-term scalability.

Technologies: Kotlin, Jetpack Compose, Material 3, Hilt, Coroutines, StateFlow, Room, Retrofit, Firebase (Auth, Firestore, Cloud Functions), ExoPlayer, Google Drive, Dropbox, and OneDrive APIs.
